Step 4: Run Simulation

Once the mesh is generated, you are ready to run your simulation on DesignSafe.

In the tab, select the folder that contains the case
Click to go to the parameters screen.

Each template case comes with a reasonable set of analysis parameters. However, it is highly recommended to review all parameters and adjust them as appropriate.

Parameters such as CAD file name, number or size of elements, etc., are found under 'mesh parameters' [1].

Parameters such as simulation duration, timestep, turbulence modeling and more are found under 'simulation parameters' [1].

You may click [3] at any time to save your changes to the remote filesystem.
Once done with your adjustments, click the [4] tab to begin running the simulation.
