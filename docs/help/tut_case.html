Step 2: Create a Case

There are multiple ways to create a new CFD Simulation case. With the CWE Tool you may

  1. Create a NEW case
  2. CLONE an existing case

Create a NEW case

Click on the tab.
Click on the Button
Select a folder to contain the new case
Enter a new unique name for the new case.
Select a template for the type of case. We recommend 2D simulation for your first case.
Press the button to initiate the case creation of DesignSafe-CI. Depending on network speed and work load on DesignSafe-CI, this may take a little.

After a short wait, the new case will be available for you to run. You can select it by returning to the tab.

  • Cloning a case operates similar to creating a case. However, use the 'Duplicate Case' button instead of 'Create New Case'.
