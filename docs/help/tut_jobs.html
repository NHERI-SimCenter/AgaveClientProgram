Step 5: Monitor Remote Jobs

Page under construction.

Once you have created a case, you can begin to do work.

Start from the 'Remote Jobs' screen:
Shows a list of all tasks controlled by the CWE Tool, one line per task.

Information provided includes

the Task Name
the status of your task. The most common options are:
  1. PENDING: your task is being prepared and transferred to DesignSafe-CI
  2. QUEUED: your task was received by DesignSafe-CI and is waiting in a queue for processing.
  3. RUNNING: your task is currently being processed at DesignSafe-CI
  4. FINISHED: task was completed. You will see the exit status on the task tab on the 'Parameters' screen. Logfiles will be available under the 'Results' screen.
A complete list of Agave states can be found here: http://developer.agaveapi.co/#job-submission.
the name of the app running/requested to run on DesignSafe-CI
the time stamp when this task was generated/updated. Time is given in DesignSafe-CI local time (Texas).
the Agave ID of the task (for identification and debugging)
