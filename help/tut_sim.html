Step 3: Run Simulation

Once you successfully completed Step 1: Create a Case and Step 2: Create a Mesh, you are ready to run your simulation on DesignSafe.

  1. In the tab, select the folder containing the case
  2. Notice that the buttons on the bottom, in particular is now available. Click this button to go to the parameters screen.
  3. On the left side of the screen are tabs for 'mesh parameters' and 'simulation parameters' [1]. For the 2D type of case, a valid mesh must be created befor a simulation can be run (Step 2: Create a Mesh).

  4. Each template case comes with a reasonable set of analysis parameters. However, it is highly recommended to review all parameters and adjust them as appropriate.

    Parameters are conveniently grouped by analysis stages: 'mesh parameters' and 'simulation parameters' [1]. (Further stages may be added with future releases of the CWE Workflow Tool.) Within each stage, parameters are further groupled by topics [2].

    Parameters such as CAD file name, number or size of elements, etc., are found under 'mesh parameters' [1].

    Parameters such as simulation duration, timestep, turbulence modeling and more are found under 'simulation parameters' [1].

  5. You may click [3] at any time to save your changes to the remote filesystem. Once done with your adjustments, click the [4] tab to begin running the simulation.
  6. You can check and monitor your remote analysis status under the tab.
