Step 1: Create and Upload a Geometry

The CWE workspace allows users to upload simulation geometries to run either 2D or 3D wind simulations against them.

For this version of the CWE, users can upload 'Alias Mesh' (.obj) files, easily creatable by FreeCAD and potentially other programs.

FreeCAD, (not affiliated with SimCenter) is freely available at: https://www.freecadweb.org

We recommend using the 'Part Design' toolkit. After creating a geometry file to test, export it as a .obj file:

  • Select the object to export.
  • Go to: File->Export
  • At the bottom, choose the file type: Alias Mesh(.obj)
  • Select a local location for this file and click 'save'.
  • Now, open the CWE program and login
  • Click on the 'Remote Files' tab
  • On the left side, find your geometry file
  • On the right side, choose a remote folder to upload to
  • Click 'upload' to upload the geometry file. Please Wait.

Once your geometry file appears among the remote files, it is uploaded and can be used to create simulations.

Please note: this software is under continual development. The experience of our testing users may prompt us to include other file upload types. We welcome constructive feedback on this matter.
