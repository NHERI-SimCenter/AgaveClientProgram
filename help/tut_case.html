Step 1: Create a Case

There are multiple ways to create a new CFD analysis case. With the CWE Tool you may

  1. Create a NEW case
  2. CLONE an existing case

Create a NEW case

  • Click on the tab
  • Click on the 'Create New Case' Button
  • Select a folder to contain the new case
  • Type a name for the new case.
  • Select a template for the type of case. We recommend 2D simulation for your first case.

After a short wait, the new case will be available for you to run. You can select it in the 'Simulation Cases' tab.

CLONE an existing case

  • Click on the 'Simulation Cases' tab
  • Click on the 'Create New Case' Button
  • Select a folder to contain the new case
  • Type a name for the new case.
  • Select a template for the type of case. We recommend 2D simulation for your first case.
