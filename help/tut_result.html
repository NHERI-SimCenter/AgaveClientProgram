Step 4: View Results and Download

Once your case has finished running, you can view results.

  1. Select the folder containing the case in the 'Simulation Cases' tab.
  2. Notice that the buttons on the bottom, in particular 'View Results/Download' is now available. Click this button to go to the results screen.
  4. This screen will show a list of possible results, depending on the type of case you have run. For each result, either the view or download icon will be available, allowing you to view or download that result. Also, at the bottom, the 'Download Entire Case' button will let you download the entire case for anaylsis.

Since this work is under development, we invite input from out users regarding the sort of results which should be made available for viewing. Further development intends to add additional entries to the result list for various templates.
