Overview

The SimCenter CWE Simulation Tool provides an interface to CFD simulations on DesignSafe-CI using template problems and/or custom designed problems of Wind Engineering. The tool manages data asynchroneous data transfer, mesh generation, queueing and runtime control of CFD simulations, and data collection. Key features:
