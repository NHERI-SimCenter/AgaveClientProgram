CWE Computational Fluid Dynamics Tool

The CWE Computational Fluid Dynamics Tool, developed by SimCenter, allows users to perform industry-grade CFD simulation, using OpenFOAM, on the powerful computers at the Texas Advanced Computing Center, through the Agave interface maintianed by DesignSafe.

You can suggest changes and post bugs by submitting issues to our code repository

Please note: this software is under continual development. We welcome constructive feedback and appriciate the patience of our testing users.
